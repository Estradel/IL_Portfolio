Groupe Imagerie @ HE-Arc
Les domaines de compétences du groupe Imagerie incluent l'acquisition, l'analyse et le traitement d'images d'une part; la synthèse d'images 3D et l'animation 3D (infographie) d'autre part. La maîtrise de ces différents aspects de l'image nous permet de traiter des problématiques variées utilisant notamment la réalité augmentée, la réalité virtuelle ainsi que les Serious Games.

1242.2 Langage C++ @ HE-Arc - 1ère année, semestre de printemps, 4.5h / semaine
Ce cours est une introduction au langage C++ et à la programmation orientée objet (POO). Il aborde les notions de classes, objets, surcharge des opérateurs, héritage, polymorphisme, transtypage, RTTI, modèles, exceptions et la bibliothèque standard. Les étudiants mettent en pratique ces concepts à travers un projet de développement d'un jeu vidéo (RPG).

3292.2 Outils d'infographie @ HE-Arc - 3ème année, semestre de printemps, 2.25h / semaine
Ce cours est une introduction aux outils d'infographie 3D. Il aborde entre autres les notions de caméra, lumière, matériaux, shaders, animation 3D, post-traitement, ombres, lightmaps, script, coroutines. Les étudiants mettent en pratique ces concepts à travers un projet de développement d'un jeu vidéo en 2D ou en 3D avec Unity URP.

2255.1 Algorithmes et structures de données III @ HE-Arc - 2ème année, semestre de printemps, 3h / semaine
Ce cours présente les algorithmes et structures de données avancés. Il aborde entre autres les notions de graphes (représentations, parcours, algorithme de Prim, Dijkstra, Ford-Fulkerson, tables de hachage. Les étudiants mettent en pratique ces concepts à travers un projet de développement en C++.

{{ else if eq $topic "DOTNET" }}

2255.3 .NET @ HE-Arc - 2ème année, semestre de printemps, 3h / semaine
Ce cours est une introduction à la plateforme .NET et au langage C#. Il aborde les notions de Common Language Runtime (CLR), Common Intermediate Language (CIL), compilation .NET, assembly, packages NuGet, types valeurs et références, espaces mémoires stack et heap, Windows Forms, WPF, XAML, contrôles graphiques, layout panel, data binding, data template, pattern MVVM, expressions lambda et LINQ. Les étudiants mettent en pratique ces concepts à travers un projet de développement d'une application en C#.
